### 👩‍💻 About Me

I'm a final-year Computer Science undergraduate at **IMS Engineering College, Ghaziabad**, focused on full-stack engineering with the **MERN stack**, and increasingly drawn toward **AI-driven and voice-first systems** — from LLM-powered code analysis to real-time conversational agents built on Retell AI and Twilio. Alongside product work, I'm building a habit of thinking about **secure coding practices** rather than treating security as an afterthought.

I've worked in a founder's-office capacity at an early-stage startup, interned on a MERN full-stack track, and taken on freelance web development engagements — so I'm equally comfortable owning a feature end-to-end or scoping a project from a client conversation.
